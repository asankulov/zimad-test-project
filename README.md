## Description
ZIMAD Test Project with Node.js, Express.js and MySQL

##Prerequisites
1. installed node.js >= 10
3. mysql database

## Installation

```bash
$ npm install
```

## Running the app

1. edit .env file from example by adding database credentials and so on

```bash

# development
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test
```
